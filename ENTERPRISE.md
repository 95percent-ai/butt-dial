# Butt-Dial MCP — Editions

## Community Edition (Free, Open Source)

**License:** MIT

Everything you need to run your own communication infrastructure for AI agents.

**Includes:**
- All MCP tools (send SMS, make calls, email, WhatsApp, voice AI)
- All provider adapters (Twilio, Vonage, Resend, ElevenLabs, etc.)
- Multi-tenant organization isolation
- Compliance engine (TCPA, GDPR, CAN-SPAM, DNC)
- Admin dashboard, setup wizard, Swagger UI
- Demo mode for safe testing
- SQLite database (local)

**You handle:** Hosting, Twilio account, provider credentials, maintenance, scaling.
