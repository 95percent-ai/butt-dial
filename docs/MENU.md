# Menu

1. **Save progress** — Write current work to all existing docs files
2. **Run tests** — Execute all test scripts and show results
3. **Run project** — Install dependencies, build, and start the project
4. **Manage permissions** — Estimate permissions needed for the next phase to minimize interruptions
5. **Sync documents** — Check that all docs are up to date and aligned with code
6. **Load progress** — Load the information required to commence operations
7. **Framework Help** — Show introduction and overview of the framework
